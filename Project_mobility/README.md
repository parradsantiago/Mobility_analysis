# Mobility analysis - LATAM 🚗💨

This project performs a correlation analysis between **urban traffic congestion (TomTom Index)** and **GDP per capita (OECD/LADB)** across various Latin American cities during 2024.

## 📊 Project Overview
The primary goal is to identify how urban mobility patterns impact regional economic productivity using Python for data cleaning and integration.

## 🛠️ Tech Stack
* **Python 3.14**: Core processing.
* **Pandas & Matplotlib**: Data manipulation and visualization.
* **SQL**: Initial data extraction.

## 📂 Structure
* 'Mobility_analysis_LATAM.py': Main script.
* 'mobility_economy_project.ipynb': EDA Notebook.
